Css - Selectores:

Selector Semantico: p {color:blue; font-size: 12px;}

Selector Id:

#selector {color:blue; font-size: 12px;}

Selector de Clase:

.selector{color:blue; font-size: 12px;}

Selector Universal: (Aplica a todos los elementos)

* {color:blue; font-size: 12px;}

Seleccionar de varias elementos:

h1,h2,h3 {color:blue; font-size: 12px;}

Selector Descenciente:

body, p, .ultimoParrafo {color:blue; font-size: 12px;}

Selector >:

div > a {color:red;} Cambiara el color de las anclas dentro del div unicamente (Hijos dentro del div)

Selectores multiples:

p {text-align:center; color:red;} p {color:gray;} Toma el alineamente y rojo - Luego lo cambia a color gris

Jerarquia y especificidad de selectores

* = 0 Semanticos = 1 Clases = 10 Id = 100 En linea = 1000

El estilo aplicará la ultima hoja linkeada si tiene la misma jerarquia y la misma especificidad

!important - Anula las otras reglas y le da prioridad a esa. p{ background-color: red !important; } #parrafo{ background-color: blue; }

Height - Width (Alto y ancho)

Nos permite modificar el ancho y alto de un elementos

Valores que puede tomar esta propiedad

auto: es un valor predeterminado, el navegador calcula la altura y el ancho length: (Longitud) define alto/ancho en px, cm, etc %: define alto/ancho en porcentaje al bloque del Contenedor initial: establece alto/ancho en su valor predeterminado inherit: (heredar) alto/ancho se heredara el valor de su elemento padre. max-height/min-height: Alto maximo y minimo max-width/min-width: Ancho maxiximo y minimo

Box-sizing

Indica como se debe calcular el tamaño de un elemento (alto/ancho) Nos permite incluir o no el padding y el border en el acho total de un elemento

Propiedades

content-box: (por defecto) no incluyen el border ni el padding en el alto y ancho border-box: el border y el padding se incluyen dentro del alto y ancho

Css-Texto

Propiedades

Estilos del tipo de letra Estilos de disposicion del texto

Tipos de letra

font-size: tamaño de la fuente (px,em,etc) font-style: estilo de la fuente (normal, italic, oblique) font-weight: ancho de la fuente (normal, bold) font-family: especifica el tipo de fuente(customizadas de google font) color: color de la fuente text-transform: (uppercase, lowercase, capitalize, none) text-decoration: (none, underline, overline, line-through) text-shadow: aplicar sombra a los textos, se especifican 4 valores (horizontal, vertical, desenfoque, color)

Disposicion de texto

Afectan al espaciado y otras caracteristicas relativas a la disposcion de texto (Espacio entre lineas, letras y como se alinea el texto dentro del div) text-align: forma en que el texto se alinea dentro (left,right,center,justify) line-height: Interlineado (multiplicador de font-size) letter-spaccing: espacio entre letras (unidades de medida) word-spacing: espacio entre palabras

Css - Pseudo clases

Se proceden por ":" Se utilizan para determinar que elemento o propiedad modifica cuando se acciona sobre un elemento Empty: Elementos que no tengas hijos :empty p: empty{ display: none; } First-child: :first-child selecciona el primer elemento hijo Last-child: :last-child seleciona el ultimo hijo Nth-child: :nth-child(N) indicar el determinado elemento hijo con la N (numero/indice) :nth-child(2n): Pares :nth-child(2n+1): Impares :nth-child(5n): Salto de 5 First-of-type: Primero de su tipo :first-of-type Last-of-stype: Ultimo de su tipo :last-of-type Nth-of-Type: Aplica con n :nth-of-type(n) Link: Enlaces que no han sido visitados aun :link Visited: Enlaces que ya fueron visitados (clickeados) :visited a:link {color:darkblue;} a:visited {color:gray;} Focus: Puede ser entrada de datos de form, click o toca elemento o seleciona con tab :focus Hover: mouse arriba :hover Active: cuando clickea :active

Pseudo Elementos

Partes especificas de un elemento Son procedidos por "::" First-Letter: Crear regla para primera letra ::first-letter First-Line: Crear regla para primera linea ::first-line Before: Primer hijo del elemento ::before After: Ultimo hijo del elemento ::after
