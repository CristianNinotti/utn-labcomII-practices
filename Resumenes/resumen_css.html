Overflow

Controla lo que sucede con el contenido que es demasiado grande para entrar en una area. Propiedades: Visible: (Por Defecto) El desbordamiento es visible. Hidden: se oculta el desbordamiento Scroll: el desbordamiento se oculta pero permite scrollear para verlo Auto: similar a scroll pero adiciona una barra de desplazamiento para scrollear

Posicionamiento

Position especifica como un elemento se posiciona en un documento. offset: top, right, bottom, left (Ubicacion final) position: static (Por defecto) no está posicionado position: relative - Se comporta igual que el static a menos que agregamos las offset causando un reajuste para adaptarse acvualquier hueco. position: absolute - se posiciona en relacion a su ancestro mas cercano, si no lo tiene, usará el body como ancestro. position: fixed - respecto a la ventana del navegador, se mantiene en el mismo lugar de la pagina donde estaba ubicado scrolleando inclusive. position: sticky - se posiciona segun la posicion de desplazamiento del usuario. Se "Pega" en su lugar de alcanzar una posicion determinada.

Posicionamiento - z-index

Con position y offset: 2 dimensiones. Z-index modifica la superposicion de los elementos. El elemento con valor z-index mas alto se mostrara adelante que los que tengan menos valor. Valores: 0 a 10 - (Por defecto: 0) div1{ z-index: 1 } div2{ z-index: 2 }

display:inline - No ocupan el 100% y si colocamos varios elementos en linea los pone un del lado de otro display: none - Se oculta el elemento display: flex - Se comporta como 1 elemento de bloque, su contenido es flexbox. Ayuda a distribuir el espacio entre items y mejora la alineacion de los mismos en diferentes tamaños de la pantalla. display: grid - Se comporta como 1 elemento de bloque y su contenido es cuadricula. FlexBox - FlexDirection

flex-direction: colum (Apila verticalmente los elementos flex-items de arriba hacia abajo) flex-direction: colum-reverse (Apila verticalmente los flex-items de abajo hacia arriba) flex-direction: row (Apila los elementos horizontalmente de izquierda a derecha) flex-direction: row-reverse (Apila los elementos horizontales de derecha a izquierda)

Flexbox - Flex-wrap

Define si permanecen en una linea o pueden fluir en otras. flex-wrap: nowrap (Por defecto) Se mantienen en la misma Linea flex-wrap: wrap (Permite que se acomoden en varias lineas por el tamaño y tamaño de pantalla de dispositivo) flex-wrap: wrap-reverse (Igual que el anterior pero se acomodan de abajo derecha hacia arriba izquierda)

Flexbox - Flexflow

Permite establecer propiedades de flex-direction y flex-wrap al mismo tiempo flex-flow: row nowrap (Por defecto) flex-flow: row wrap (No los apila, los acomoda en base a la pantalla de dispositivo)

Flexbox - Justify-Content

Permite establecer como se destribuyen los item-flex en el elemento padre justify-content: flex-start (Por defecto) Ordena los items uno al lado de otro a la izquierda. justify-content: flex-end - Los alinea a la derecha justify-content: center - Al Centro. justify-content: space-around - deja un espacio a los bordes y entre si. justify-content: space-between - distribuye los elementos en si, sin espacio entre bordes

Flexbox - Align-Items

Permite alinear verticalmente los items. align-items: center - Al medio del contenedor. align-items: flex-start - En la parte superior izquierda. align-items: flex-end: En la parte inferior izquierda. align-items: stretch - Estira los elementos para rellenar el alto del contenedor. align-items: baseline: Alinea los elementos como sus lineas de base se alinean. Grid

Grid ofrece un sistema de diseñado basado en cuadriculas. Filas y columnas basicamente. Facilita el diseño para el posicionamiento. display: grid (Para convertir el contenedor en grilla) Una lista con 6 items .div{ display: grid; grid-template-columns: auto auto auto; } .div2{ item1 item2 Item3 ... } Los acomoda en 2 filas y 3 columnas los 6 items

Grid - Componentes

Columnas: grid columns: lineas verticales Filas: grid rows: lineas horizontales Espacios: grid gaps: espacios entre columnas y filas (Se puede ajustar con propiedad grid-gap) Lineas: grid lines: Son las lineas entre cada columna y filas. Referencia de cuanto ocupa cada elemento. Grid Area: Al mismo tiempo, podremos definir un área o junto de celdas y ubicarlas en nuestra grilla. (No entendi)

Grid - Propiedades

grid-template-columns: Define columnas en grilla y ancho. grid-template-rows: Define la altura de cada fila. El valor de las filas es definido por los items y las columnas de columns. Posibles propiedades para ambos templates: none: no hay cuadricula explicita absoluta: no negativa (10px, 2cm) relativa: no negativa (20%, 3rem) auto: Ocupa espacio de manera automatico, se distribuyen de mayera equitativa repeat(n, val): Establecer valores petidos, en n el valor de veces a repetir y el valor a repetir. repeat(6,auto) <-- Efectivo (Es lo mismo que auto auto auto auto auto.) grit-template-areas: Nos permite organizar los elementos por nombres. grid-template-areas: header header header main main aside footer footer footer; grid-area: Nombre que le asignamos al area de nuestras grilla. Propiedades de area: none: no hay grid en areas definidas. nombre de grid area: nombre definido para el elemento hijo de grid area. . (punto): significa celda vacia.

Grid - Propiedades - Grid - Items:

grid-column-start: linea donde la columna comienza. grid-column-end: linea donde la columna termina. grid-column: propiedad abreviada para ambas anterior. Se colocan valores start y end [start]/[end] grid-row-start: linea donde la columna comienza grid-row-end: linea donde columna termina grid-row: propiedad abreviada para ambas anterior. Se colocan valores start y end [start]/[end] justify-self: alinea un grid item dentro de una celda a lo largo de la fila. Propiedades: stretch (Por defecto), start, end, center. align-self: alinea un grid item dentro de una celdda a lo largo de la columna. Propiedades: stretch (Por defecto), start, end, center. FlexBox vs Grid

Flexbox: 1 dimension. Para componentes. Grid: Multidimensiones. Para layouts.

Media Queries

Consultas de medio (En español) son para modificar tu pagina web o apps, en funcion al dispotivo con cual es ejecutado y ajustar su pantalla al mismo. (celular, tablet, pc escritorio), se puede modificar caracteristicas y propiedades de todos los elementos. Por lo general anchos y demas. Viewport. Verificar: Ancho y alto de la ventana grafica. Ancho y alto del dispositivo. Orientacion (Por dispositivo, vertical u horizontal) Resolucion. Se resuelve por un verdadero y falso, cuando resulte verdadero se aplica Ejemplo: @media not|only mediatype and (expressions) { CSS-CODE; } Tambien puede tener difentes hojas de esitlo pasra diferentes medios: Mejor ejemplo: @media screen and (max-width: 480px){ h1 { font-size: 25px; color: brown; } } Toda la pagina es normal hasta que se abre con un celular con maximo de 480px aplica esos efectos en este h1.
